## Getting Started

This application uses:
- Ruby 2.7.4
- NodeJS (v16), and npm
- Heroku CLI
- Postgresql
- Bootstrap

Gems used:
- email-validator
- devise
- fast_jsonapi
- devise-jwt

npm packages used:
- react-datepicker
- react-moment

Run
```
bundle install
npm install --prefix client
```
to have all the necessary gems and packages.

You may use the db/seeds.rb file to seed the project.
